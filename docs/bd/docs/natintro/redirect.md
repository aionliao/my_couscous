## 点击下载 ##
点击下载，返回下载的真实地址

## 请求说明 ##
### URL ###
> 请求应用数据中提供的下载地址（即字段download_url值）即可

### 请求方式 ###
> GET

### 请求参数 ###
无

## 返回结果 ##
```php
header("Location:http://a.gdown.baidu.com/data/wisegame/afdf91d34207d08a/huanledoudizhuHD_3439.apk");
```
